# Tag Cloud

### Console version

```
positional arguments:
    -i, --input        Input data path
    -o, --output       Output path

optional arguments:
    --help             Display this help screen.
    --version          Display version information.
    -w, --width        (Default: 1200) Image width    
    -h, --height       (Default: 1200) Image height
    -a, --algo         (Default: Circular) Layout algorithm type
    -t, --theme        (Default: Rainbow) Coloring theme
    -f, --font         (Default: Arial) Tags font
    -e, --extension    (Default: png) Image format
    -c, --count        (Default: 40) Words count
```
