# Генератор облака тегов

 С помощью данной утилиты можно генерировать облака тегов по указанному источнику слов.

# Быстрый старт

## Поддерживаемые команды

  **--input**          [Обязательный параметр] Задаёт файл-источник, который содержит текст, для построения облака тегов

  **--output**         [Обязательный параметр] Задаёт файл-результат, который будет представлять собой сгенерированную картинку, содержащую облако тегов

  **--width**          [Обязательный параметр] Устанавливает ширину картинки выходного файла

  **--height**         [Обязательный параметр] Устанавливает высоту картинки выходного файла

  **--stopwords**      Задаёт файл, содержащий слова, которые будут игнорироваться

  **--background**     Устанавливает цвет заднего фона

  **--layouter**       Задаёт тип алгоритма, по которому будет сгенерировано облако тегов. Доступные алгоритмы: ArithmeticSpiral (по умолчанию), SquareSpiral

  **--colorscheme**    Задаёт цветовую схему, по которой будут раскрашены слова облака тегов. Доступные схемы: RandomColors (по умолчанию), TransparentRed

  **--fontscheme**     Задаёт схему шрифтов, в соответствии с которой будут отображаться слова облака тегов. Доступные схемы: Arial (по умолчанию)

  **--sizescheme**     Задаёт схему размеров, в соответствии с которой для каждого слова будет задан его размер. Доступные схемы: Linear (по умолчанию)

  **--help**           Отображает справку по использованию программы

  **--version**        Отображает информацию о версии программы

  **--ignoreboring**   Указывает программе, стоит ли исключать из обработки "скучные слова" (предлоги и т.п.). true (по умолчанию) или false

## Примеры использования
 
 Команда: TagCloudCreator.exe --input d:\input.txt --output d:\output.png --stopwords d:\stopwords.txt --width 5000 --height 5000 --colorscheme TransparentRed
