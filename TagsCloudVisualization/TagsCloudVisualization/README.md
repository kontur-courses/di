### Запуск приложения
_Вам нужно заполнить пять переменных_

* Размер изображения `-imageSize`
* Путь до файла со словами `-filePath`
* Цвет слов в облаке `-color`
* Название шрифта `-fontName`
* Форма кривой `-curve`


Кривая может быть - `spiral | heart | astroid`

Пример - `.\TagsCloudVisualization.exe -count 450 -maxLength 20 -curve heart`
