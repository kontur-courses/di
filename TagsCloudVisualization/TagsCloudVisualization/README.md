### Запуск приложения
_Вам нужно заполнить три переменные_

* Количество прямоугольников `-count`
* Максимальная ширина прямоугольника `-maxLength`
* Форма кривой `-curve`

Кривая может быть - `spiral | heart | astroid`

Пример - `.\TagsCloudVisualization.exe -count 450 -maxLength 20 -curve heart`
