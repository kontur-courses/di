Примеры запуска:
args = {"--help"} Выведет все возможные клю для приложения
args = {"-f", "timeSolution.txt"} Создаст дефолтно настроенное облако из слов в timeSolution
args = {"-f", "timeSolution.txt", "-c", "Red", "-b", "Black"} Красные слова на чёрном фоне
args = {"-f", "timeSolution.txt", "-h", "100", "-w", "100"} Квадратное изображение 100x100
Аргументы можно засунуть через конфиги
